# Tower v2

Enterprise-портал для сотрудников Artkai. Переписан с нуля, см. [PLAN.md](PLAN.md).

## Quickstart

```bash
yarn
yarn dev
```

## Команды

| Команда | Что делает |
|---|---|
| `yarn dev` | Vite dev server |
| `yarn build` | Production build |
| `yarn lint` | ESLint |
| `yarn typecheck` | TypeScript check |
| `yarn test` | Vitest |
| `yarn check` | lint + typecheck + test |
