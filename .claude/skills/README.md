# Skills

Scaffolding. Each skill = a folder with `SKILL.md` (frontmatter `name` + `description`, sections: Trigger / Inputs / Pre-flight / Steps / Gotchas / Templates / Validation loop / Post-flight).

Skills are written **while it's fresh, after real hands-on work** — see [`../../docs/skills.md`](../../docs/skills.md) for the full inventory (Tier 1+2 table), principles, methodology, and the `SKILL.md` template.

Right now the folders are scaffolded but the `SKILL.md` files are empty — they get filled in the first time a pattern is applied.
